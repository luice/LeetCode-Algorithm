## 题目

给你两个非空链表，表示两个非负整数。每个节点代表一位，数位逆序存储。把两个数相加，并返回一个链表。

除了 0 这个数本身外，给定的数没有前导 0。

例：

输入：`(2 -> 4 -> 3) + (5 -> 6 -> 4)`

输出：`7 -> 0 -> 8`

说明：`342 + 465 = 807`.

## 解题思路

本题考的是链表的基本操作。

没说数的范围，所以错误的做法是先转换为 int 或 long 然后算然后转换为链表，因为很可能会超时。
正确的做法是，像手算加法一样，一位一位地算，建一个新链表，一位一位地，一边算一边把结果存进去。
需要注意的是，两个一位数相加，结果可能是十几，所以要维护一个进位，参与下一位的加法运算，这一过程与手算一样。
还要注意，给定的两个链表可能不一样长，要注意空指针的问题。当一个链表算完以后，要接着算另一个链表。注意是算而不是简单复制，因为有进位的问题。最后，如果需要，也要把进位存进一个节点。
